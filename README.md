# EduCake boy
<br>
EduCake boy 是一個在 86Duino EduCake上使用 L86duntu作業系統的工具程式，使用 USB Display 做的一個小型機台。
<br>
[簡介影片](https://www.youtube.com/watch?v=6gJqxwAcF7s)
<br>
EduCake boy除了把原始碼公開以外，也將[建構教學](http://roboardgod.blogspot.tw/2017/01/educake-boy.html)也一併公開，任何人都可以自由使用、修改、散佈。

## I. File list
-------------
	1. \arcade.py
		這是利用 mraa 搭配 python-evdev 來將外部訊號轉換為鍵盤訊號，以達到模擬機台的效果。
		
	2. \autorun
		這個script可以讓我們在開機時自動啟動所有程式，至於模擬器以及遊戲ROM檔案請自行添加。
